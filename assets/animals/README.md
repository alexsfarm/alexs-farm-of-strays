# Animal photos

Drop each animal's photo in this folder using the EXACT file name below
(lower-case, .jpg). The website already points at these names, so once a file
is here it appears on the "Looking for a home" cards automatically. Until a
photo is added, that card shows a friendly placeholder.

| Animal  | File name              |
|---------|------------------------|
| Orion   | `orion.jpg`            |
| Alice   | `alice.jpg`            |
| Wilma   | `wilma.jpg`            |
| Aeris   | `aeris.jpg`            |
| Eleni   | `eleni.jpg`            |
| GeeGee  | `geegee.jpg`           |
| Kittens | `kittens.jpg`          |
| Maddox  | `maddox.jpg`           |

Tips: landscape photos crop best (cards are 4:3). Any size is fine — the card
crops automatically. To add a new animal, add its entry in `config.js` and put
its photo here with a matching file name.
